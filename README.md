# CSE Data Science & MCA Department Portal

A comprehensive department portal for Computer Science & Engineering students, faculty, and administrators.

## Features

- **Public Sections**: Home, About, Faculty, Projects, Events, Toppers, Contact
- **User Authentication**: Login/Register with email verification
- **Role-based Dashboards**: Student, Faculty, and Admin dashboards
- **Theme Support**: Light/Dark mode toggle
- **Responsive Design**: Mobile-friendly layout
- **Real-time Data**: Dynamic content loading from backend API

## Tech Stack

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript (ES6+)
- AOS Animation Library
- Chart.js for analytics
- Font Awesome Icons
